# 📚 Estudos

Repositório dedicado aos meus estudos de desenvolvimento de software utilizando C# e .NET.

O objetivo deste repositório é documentar minha evolução técnica através da implementação de exemplos práticos, exercícios e projetos voltados para Programação Orientada a Objetos, princípios de desenvolvimento, arquitetura de software e boas práticas.

---

# 🚀 Objetivos

* Fortalecer os fundamentos de desenvolvimento com C# e .NET
* Aprender e praticar Programação Orientada a Objetos (POO)
* Aplicar os princípios SOLID
* Estudar Design Patterns
* Compreender arquiteturas modernas de software
* Desenvolver código limpo, legível e manutenível
* Construir uma base sólida para desenvolvimento backend

# 📐 SOLID

Aplicação dos cinco princípios para construção de software flexível e manutenível:

* SRP — Single Responsibility Principle
* OCP — Open/Closed Principle
* LSP — Liskov Substitution Principle
* ISP — Interface Segregation Principle
* DIP — Dependency Inversion Principle
